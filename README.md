# USB-RubberDucky
USB Rubber ducky which injects a basic keylogger into the target's Device using Arduino UNO

You'll have to convert the "keylogger.py" file into an exe, host it on some server, and then feed that URl in the rubberduckyscript.ino file. Currently the site it is pulling this keylogger from is not active, so it wont work.

You will also have to flash your arduino with the Arduino-Keyboard-0.3.hex file.
